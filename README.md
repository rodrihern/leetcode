# LeetCode

Practice problems organized by problem number and title.

## Structure

```text
/problems
  /0042_trapping_rain_water
    description.md
    solution.py
```

Each problem lives in its own directory. LeetCode problems use:

- `{number}_{problem_slug}` when the official number is known
- `{problem_slug}` for problems without a LeetCode number
